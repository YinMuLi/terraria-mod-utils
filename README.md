## 使用
使用了NuGet，因此无法使用tModLoader进行编译，需要使用IDE进行编译

使用IDE，如Visual Studio 2022、Rider等打开项目
如果先前在tModLoader中启用了此Mod，先禁用，然后重新加载
使用IDE编译项目
在tModLoader中打开此Mod，重新加载
成了
